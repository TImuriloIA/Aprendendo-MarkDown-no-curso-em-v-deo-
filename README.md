## Formatações Simples


 **Aprendendo linguagem Markdon no curso em vídeo 2025**



Marcação em negrito (**) ou (__) SEM ESPAÇO

EX:


(**) Aprendendo linguagem **Markdon** no curso em vídeo em 2025


(__) Aprendendo linguagem __Markdon__ no curso em vídeo em 2025


Italico (*) ou (_)



(*) Aprendendo linguagem *Markdon* no curso em vídeo em 2025


(_) Aprendendo linguagem _Markdon_ no curso em vídeo em 2025


Texto Tachado  (~~)

EX:

(~~) Aprendendo linguagem  ~~Markdon~~  no curso em vídeo em 2025



Título (#) nível I - Título Principal
(##) subtítulo 
Conforme for adcionando as '##' sucessivamente , teremos subtítulos no código 


Linha (---) ou (***)

ex (---):

---

ex (***):

***




### Misturando Configurações 


Podemos ( __ )( * ) __*misturar*__ configurações 

***

#### Lista Númerada

(1.) Teste

(1.) Teste2 

(1.) Teste3 

Ex:

1. Teste
1. Teste2
1. Teste3


Podemos também torna uma lista sendo subitem de outra em números Romano



(1.) Teste

(3 espaço) (1.) Teste2 

(1.) Teste3 

EX:

1. Teste

   1. Teste2 

1. Teste3

***

Lista demarcada (*) ou (-)
 

( *  Arroz )
( *  Feijão )
( - Alface )
( - Abacaxi )

EX:

* Arroz
* Feijão
- Alface
- Abacaxi 



Lista de TAREFAS ( - [ ] )

( - [ ] Criar um Portifólio)
( - []  Criar um canal no YouTube)
( - [] Criar um Instagram Comercial )

EX :

 - [ ] Criar um Portifólio
 - [ ] Criar um canal no YouTube
 - [ ] Criar um Instagram Comercial 

Ao realizar a tarefa, basta por o 'x' :
 (- [x] Criar um Portifólio)
 (- [ ] Criar um canal no YouTube)
 (- [ ] Criar um Instagram Comercial)

EX:
 - [x] Criar um Portifólio
 - [ ] Criar um canal no YouTube
 - [ ] Criar um Instagram Comercial
